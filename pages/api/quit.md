ASCII Command Line Interface (CLI)を終了します



## API パラメータ

パラメータはありません



## API Call テンプレート

```
quit
```



### JSONのSuccess Response

```
{
"data":null,
"description":"Bye!",
"status":"SUCCESS"
}
```



#### JSON Response

JSON responseは以下を含みます:

- data – パースするデータはありません
- description– コマンドのパース・実行結果
- status – コマンドがパースされ正常実行された場合は**SUCCESS** そうでなければ**FAIL**
